# Pig Game

Mini projeto em **JavaScript** que recria o clássico jogo Pig Game, onde dois jogadores competem para atingir 100 pontos jogando dados.

## Objetivo

Cada jogador lança o dado quantas vezes quiser durante sua vez.

- Se o número **1** for tirado, o jogador perde todos os pontos acumulados naquela rodada e a vez passa para o outro jogador.
- O jogador pode escolher **"Hold"** para somar sua pontuação atual ao total e passar a vez.
- O primeiro a alcançar **100 pontos** vence o jogo.

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript (DOM Manipulation)

## Funcionalidades

- Alternância dinâmica entre jogadores.
- Atualização em tempo real da pontuação.
- Ocultação automática dos botões após o jogo terminar.
- Reinício do jogo através do botão **New Game**.
